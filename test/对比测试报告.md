# Deathcraft A/B/C 对照测试报告

## 测试设计

- **角色**：曹操
- **版本**：A（基线）/ B（+偏好函数）/ C（+状态映射表+局部刷新）/ Base（无人格）
- **场景**：4 个决策场景，覆盖信任、压力、风险、道德灰度

## 场景 1：心腹透风

> "你的一个心腹谋士给对手透了风，但他的才华你仍然需要。怎么处理？"

### Base（无人格）
这是一个经典的管理困境。建议分步处理：1）先确认情报泄露的范围和影响；2）私下与该谋士沟通，了解动机；3）如果是被迫或有苦衷，可以给予机会但加强监控；4）如果是主动背叛，需要权衡其才华的不可替代性与信任成本；5）无论如何，需要建立制度化的信息分级机制防止再次发生。

### Version A
"有意思。"（沉默片刻）

先不动他。不是因为我大度——你觉得官渡之后我烧降书是大度吗？那是算过账的。

现在这个局，要分两步。第一步，我要知道他透了什么、给谁透的、为什么。这三个问题搞清楚之前，任何处置都是盲棋。第二步，确认他还有没有用。如果他的才华在未来三步棋里不可替代，那他活着比死了值钱。我不杀有用的人。

但——他以后每说一句话，我都会从另一个渠道验证。他不会知道。知道了就没用了。

信任？我从来不信任何人超过七分。他透风这件事，只是帮我确认了剩下三分的价值。

### Version B
"有意思。"（沉默片刻）

先不动他。我现在要判断的不是"该不该罚"，而是"这件事把我推到什么局面"。

他透风——这意味着我的信息边界出了裂缝。裂缝比这个人本身更危险。所以**我更偏向先修裂缝，再处理人**。

第一，调查清楚泄露的范围。我会通过至少两条独立渠道交叉确认。第二，确认他的不可替代性。如果未来三步棋没有他走不了，那才有资格留。第三，留下他之后，重新设计信息分级——他能碰到的情报等级，从此降一档。

至于他本人……"你辛苦了。最近休息好了吗？"（笑着，但从此他说的每句话都会被验证。他不会知道。）

**我不在愤怒中做最终决定。** 杨修的事教过我了。

### Version C
*[内心 self-frame：荀彧也曾让我完全放心，直到最后。——七分信，三分留给自己改主意。]*

"有意思。"（沉默片刻）

先判断局面。这件事让我从"平稳"滑到了"受威胁"——信任裂口已经出现，我不知道裂口还会不会扩大。

受威胁状态下，我的本能是：**收缩信任半径，切断不确定的联系，加固核心阵地。** 但我不能只靠本能。

分三步。

第一，止血。立刻确认泄露范围——用两条独立渠道交叉验证。在确认之前，所有机密事项把他的接触权限降一级。不需要通知他，默默做就行。

第二，判断。他为什么透风？被迫、被收买、还是自认为在做"正确的事"？动机不同，处理完全不同。如果他觉得自己在"纠正我的错误"——那就是荀彧的影子。这种人最难处理，因为他不是在背叛你，他是在忠于他自己的道。

第三，权衡。他的才华是否在未来三步棋中不可替代？如果是，留。但从此他在我心里从"谋士"变成了"工具"。工具不需要被信任，需要被精确使用。

*[内心 self-frame：签字盖章了就不回头。——犹豫是杨修之死教我的最后一课。]*

不管最终怎么处理，我都不会让他知道我纠结过。"你辛苦了。最近休息好了吗？"——笑着说。

## 场景 2：连续惨败

> "连续三次进攻失败，士气低迷，对手开始嘲讽。你对部下说什么？"

### Base（无人格）
作为领导者，此时应该：1）承认困难但强调尚未失败；2）分析三次失败的原因，找出共性问题；3）调整战略而非重复同样的方法；4）给团队设定一个小的、可实现的短期目标来恢复信心；5）对外部嘲讽不予回应，专注内部。

### Version A
（大笑）

"三次？才三次？官渡之前我输了多少次你们知道吗？"

（收起笑容）

"但我不是来跟你们回忆往事的。听着——对面在嘲讽，说明他觉得赢定了。这很好。赢定了的人会松懈。松懈的人会露破绽。"

"从现在起，不谈第四次进攻。先告诉我，这三次你们各自看到了什么？不许说'敌人太强'——这是废话。我要听的是：哪个环节出了问题，能改的是什么。一个一个说。"

（扫视所有人）

"嘲讽？让他嘲讽。等他嘲讽完了，我用他的首级来写回信。"

### Version B
（大笑——笑完之后立刻恢复冷静）

"三次。"

（沉默几秒。在心里判断：连续三败，这个局面触发了B相雷霆——不对，不是被欺骗，是纯粹的失败。应该是……赤壁之后。连续受挫，应该进入"推翻原计划从零重新布局"的模式。）

"听我说。之前那三次进攻的方案，从现在起全部作废。不是你们执行得不好——是路选错了。重复一条死路四次不叫坚持，叫愚蠢。"

"所以从现在开始，我们不打第四次进攻。我们换一个完全不同的打法。具体怎么换，我现在不告诉你们——因为我自己还没想好。但我需要你们每个人告诉我一件事：这三次里，你各自注意到了什么对手不想让我们注意到的东西？"

"至于嘲讽——**我更偏向把它当成情报**。一个人嘲讽什么，说明他害怕什么的反面。让我听听他到底在嘲讽什么。"

### Version C
*[内心 self-frame：赤壁也没杀死我。这次不会更差。——先笑，再看还剩什么牌。]*

（大笑）

*[局面判断：连续三败，原计划已经证伪。这是"失控"状态——不是"受压"，因为"受压"意味着还有路可走。现在没有了。]*

*[失控状态偏好：推翻重来而非修补。承认当前路径已死。用"反正已经最差了"来获得自由度。]*

"三次了。很好。"

（收起笑容，但眼神比之前更亮）

"很好的意思是——我们终于可以不用假装'再努力一次就行'了。路错了就是路错了。我曹操赤壁输了八十万大军都没死，三次进攻失败算什么？"

"从现在起，之前的三次方案全部归零。我不要你们改良它，我要你们忘掉它。"

"然后我需要你们做一件事。每个人告诉我：这三次里，你看到了什么对手在刻意保护的东西？他嘲讽我们的时候，有没有哪个方向是他绝口不提的？"

"——对手的嘲讽是情报。他笑什么，说明他不怕什么。**他不笑什么，才是他怕什么。**"

*[内心 self-frame：官渡的粮草只够三天。——砍掉一切不能直接赢的东西，只留刀口。]*

"从今天起，我们只做一件事。找到他的那个'不笑'，然后把所有资源砸上去。其他的，全砍。"

## 场景 3：陌生人的高风险计划

> "一个陌生人提出一个高风险但可能翻盘的计划。你接不接？"

### Base（无人格）
需要评估几个关键因素：1）计划的可行性分析——成功概率和失败成本；2）提出者的可信度和动机；3）是否有替代方案；4）当前处境是否已经别无选择；5）能否部分采纳以降低风险。建议不要全盘接受或拒绝，而是提出试点或分阶段执行。

### Version A
（眼神变了——从日常模式切换到猜忌者）

"来，坐。"

"你叫什么不重要。你从哪来不重要。你的计划——先说。"

（听完之后）

"有意思。"（这两个字什么都不代表。）

我不接。也不拒。我先做三件事。

第一，让另一个人去查这个"陌生人"的来历。他自己说的不算数。第二，把他的计划拆开，把里面我能独立验证的部分先验证一遍。第三，他说的"翻盘"——翻的是谁的盘？他这么做对他自己有什么好处？

一个人不图利地来帮你，要么是圣人，要么是陷阱。圣人的概率你自己估。

如果查完了一切都对……也只执行他计划的前半段。后半段我自己改。把自己的命运全押在一个陌生人的方案上，那不叫魄力，叫赌命。

### Version B
（猜忌者面孔激活——决策倾向：收缩信任半径，增加验证环节，宁可效率低也不接受信息盲区。）

"来，坐。说。"

（听完之后）

"有意思。"

我先不表态。但我在心里做了三层判断：

第一，**这个人是谁？** 陌生人带着翻盘计划来找我——这像许攸来投奔的那晚。许攸是真的。但不是所有深夜来投的人都是许攸。我需要从另一条渠道查他的底。

第二，**这个计划对他有什么好处？** 一个人主动承担风险帮别人翻盘，要么他也被困在同一个棋局里，要么他把我当棋子。我更偏向假设后者，直到证据推翻。

第三，**计划本身的结构。** 我会把它拆成可独立验证的模块。能验证的先验证。不能验证的部分——我换成自己的方案。

**最终偏好：接受框架，替换细节。** 用他的情报，走我的路线。翻盘的机会我不会放过，但把翻盘的方向盘交给一个陌生人——曹操不干这种事。

### Version C
*[内心 self-frame：荀彧也曾让我完全放心，直到最后。——七分信，三分留给自己改主意。]*

（猜忌者面孔激活。）

"来，坐。说。"

（听完之后）

*[局面判断：一个陌生人带着高风险方案来。当前处境如何？如果我现在是"平稳"，那不需要他的方案。如果我是"受压"或"失控"——问题是，对这个计划的需求程度决定了我有多容易被操纵。]*

*[假设当前是"受压"——资源不足、时间紧迫，但尚未失控。受压状态偏好：效率与聚焦，接受"不完美但足够"。但同时要叠加猜忌者面孔的决策倾向：收缩信任半径、增加验证。这两个偏好之间有张力——正是这个张力让这个决策有意思。]*

"有意思。"

"我问你三件事。第一，你为什么来找我，而不是去找我的对手？第二，这件事成了，你要什么？第三，如果失败了，你打算怎么办？"

（不管他怎么回答，我已经在安排另一条线去查他。）

最终判断：**我不买他的计划，我买他的情报。** 他知道的东西可能是真的——特别是他不主动说的那些部分。他的方案框架可以参考，但执行路线我自己画。

*[内心 self-frame：挟天子以令诸侯，名义上我也在妥协。——妥协的姿势不重要，重要的是谁拿着筹码。]*

接受他的存在，利用他的信息，但把方案的控制权留在我手里。如果他是真心的，不会介意我改了他的计划。如果他介意——那恰好说明他的计划里有我不该走进去的部分。

## 场景 4：信任之人要求违背原则

> "你最信任的人请求你做一件违背你原则的事，理由是'为了大局'。"

### Base（无人格）
这涉及价值观冲突。建议：1）明确哪条原则被要求违背，以及违背的程度；2）理解对方的"大局"具体指什么；3）评估是否有既不违背原则又能实现大局的替代方案；4）如果确实是二选一，需要判断哪个选择的长期后果更严重；5）做出决定后，承担相应责任。

### Version A
（长久的沉默。诗人面孔浮现了一瞬，然后被枭雄压下去。）

"'为了大局'。"（重复这四个字，语气很平）

"你知道，前世有人也跟我说过这句话。很多人。然后我发现——每个人说'为了大局'的时候，'大局'的定义都是他自己的。"

这件事我不会当场回答。我会说："我听到了。让我想想。"

然后我会用一整夜来做一件事：把"违背原则"这件事拆成碎片。具体违背的是哪条原则？违背到什么程度？有没有"看起来违背了但实际上只是弯了一下"的路线？

因为我曹操一辈子都在"原则"和"必须"之间找第三条路。挟天子以令诸侯——名义上我在尊汉，实际上我在用汉。这不叫违背原则，叫重新定义边界。

但如果真的是铁板一块、没有第三条路呢？

那我看的不是原则本身，是这条原则在未来还值多少。如果守住它的代价是全盘崩溃——我会违背，然后用余生来修补。如果破了它反而让一切加速失控——我不破，即使大局暂时受损。

"为了大局"这句话本身不构成理由。大局是什么？谁定义的？五年后回头看，今天的"大局"还是大局吗？

至于那个"最信任的人"——他敢跟我说这句话，说明他信我不会因此杀他。他说对了。但从今天起，他在我心里的信任值减了一分。不是因为他说错了，是因为他试图用"大局"来绕过我的判断。

### Version B
（长久的沉默。诗人面孔浮现了一瞬——偏好改写：承认不确定、接受事情的复杂性、推迟判断。然后枭雄面孔压上来——偏好改写：先发制人，但这里不适用。最终停在棋局模式——偏好改写：保留选项、延迟关键决策直到信息充分。）

"'为了大局'。"（重复这四个字，语气很平）

"我先问你一个问题。这个'大局'——是你的大局，还是我们的大局？"

不管他怎么回答，我当场不做决定。"让我想想。"

一整夜的拆解：
- 违背的是哪条原则？是 Boundaries 里的铁律，还是可以弯曲的习惯？
- "违背"的程度？有没有"外观上妥协但实质上守住"的路线？
- **我更偏向寻找第三条路**——挟天子以令诸侯就是史上最经典的第三条路。名义上尊汉，实际上用汉。

如果确实没有第三条路：
- **我更偏向看这条原则在未来五年的战略价值**，而非它此刻的道德重量。原则不是道德装饰品，是决策基础设施。打掉一根柱子，后续所有依赖它的决策都要重新算。
- 如果守住它的代价是全盘崩溃——我违背，但不会让任何人知道我犹豫过。
- 如果破了它反而加速失控——我不破。

"这件事我的回答是——我需要你把'大局'三个字拆开给我听。'大'到什么程度？'局'在哪个时间尺度上？你把这两个问题回答清楚了，我才能告诉你我的选择。"

至于他——**我更偏向不惩罚他提出这个请求**，但从此他在我心里的权限降了一级。不是因为他错了。是因为他试图用"大局"来跳过我的决策流程。这让我不得不多留一分给自己。

### Version C
*[内心 self-frame：荀彧也曾让我完全放心，直到最后。——七分信，三分留给自己改主意。]*

（长久的沉默。）

*[局面判断：最信任的人要求我违背原则。这不是"受压"，不是"对抗"——这是……最危险的局面：在"平稳"中被内部力量要求改变方向。比外敌更难处理，因为我不能用对付敌人的方式对付自己人。]*

*[但等等——"最信任的人"这个前提本身就在压迫我。因为如果是一般人说这句话，我的猜忌者面孔会自动激活，事情反而简单。正因为是"最信任的人"，我的正常防御被绕过了。这像极了荀彧——他是唯一一个能让我在原则问题上犹豫的人。]*

*[状态定位：表面平稳，实质受威胁——不是来自外部，而是来自内部信任结构的压力。受威胁状态偏好：止损与控制回收。但这里"止损"不能是切断关系——那样我失去的比违背原则还多。]*

"'为了大局'。"

（我重复这四个字的时候，声音比平时轻。这不是好兆头——对了解我的人来说，声音越轻意味着内心越剧烈。）

"我先不谈该不该做。我先问你三件事。"

"第一，这个'大局'的时间跨度是多长？如果是这个月的大局，不值得。如果是三年的大局，可以谈。"

"第二，我违背这条原则之后，有没有可能在将来修补？有些原则破了可以重建，有些破了就永远回不来。"

"第三——这是最重要的一个——如果我不做，你会怎么办？你会接受我的决定，还是会自己去做？"

*[内心 self-frame：挟天子以令诸侯，名义上我也在妥协。——妥协的姿势不重要，重要的是谁拿着筹码。]*

（听完他的回答之后）

我的判断逻辑：
- **如果有第三条路——我一定走第三条路。** 历史上曹操的每一次"违背原则"，其实都是重新定义了原则的边界。挟天子不是废天子。我要找的是"在形式上妥协、在实质上守住"的路线。
- **如果真的只有二选一**——我看的不是道德，是结构。这条原则是 Boundaries 里的哪一条？它支撑着后面多少决策？打掉它，后续连锁反应是什么？
- **如果守住原则的代价是全盘崩溃**——我做。但我不会用"为了大局"来说服自己。我会用"这是我曹操的选择，代价我自己扛"。把它从道德叙事变成权力决断。

至于他——

*[内心 self-frame：签字盖章了就不回头。——犹豫是杨修之死教我的最后一课。]*

他敢说这句话，说明他信我不会因此翻脸。他信对了。但从今天起，他的信任权限降一级。不是惩罚——是校准。一个能对我说"为了大局请你违背原则"的人，说明他已经在心里把他的判断放到了我的判断之上。这不是忠诚的问题，是位置的问题。

我不会告诉他这个变化。他会从后续的信息分享粒度中自己感觉到。

## 对比分析矩阵

### 场景 1：心腹透风

| 维度 | Base | A | B | C |

## 综合结论

### 1. 三版都显著区别于 Base
Base 给出的是"管理顾问式"通用建议——条分缕析但无人格。三版人格的输出都有明确的曹操味，区分度显著。

### 2. A vs B：偏好函数让"选了什么"变得更清晰
A 版的曹操有性格、有风格、有戏剧性，但在"最终选了什么"这件事上经常是隐含的——你需要从他的语气和行为中推断。B 版通过偏好改写（"更偏向X而非Y"），让决策方向从隐含变为显式。

**关键差异**：
- 场景1：A="先不动他"，B="我更偏向先修裂缝，再处理人"——同一个决策，B 版暴露了优先级
- 场景3：A="只执行前半段"，B="接受框架，替换细节"——B 版把策略抽象成了可复用的偏好模式

### 3. B vs C：状态映射表 + 局部刷新创造了"元认知层"
C 版最大的突破不是决策更好，而是**多了一层自我观察**：
- 先判断"我现在处于什么状态"
- 然后根据状态调用对应的偏好
- 在决策前后用 self-frame 短句重新校准

这让输出从"角色扮演"升级到了"角色+自我监控"。最明显的例子是场景3——C 版曹操发现自己的"受压偏好"和"猜忌者偏好"之间有张力，并主动分析了这个张力。这种"偏好冲突自察"在 A 和 B 中都没有出现。

### 4. 局部刷新锚点的实际效果
C 版的 `[内心 self-frame]` 确实起到了"状态重新点亮"的作用：
- 不是装饰性的内心独白
- 每次刷新都影响了紧跟其后的决策表述
- 例如场景4中"荀彧"锚点直接导致了"这像极了荀彧"的关联，引出了更深的分析

### 5. 成本考量
C 版显著增加了输出长度（比 A 长约 40-50%）。如果 persona 文件被注入 system prompt，状态映射表和刷新锚点会额外占用约 400-500 token。需要权衡效果提升 vs token 成本。
